➕ CalculadoraSuma

Librería Java para ejecutar sumas con datos tipo double.

📦 Distribución

Diseñada para ser exportada e integrada como archivo JAR en proyectos externos (Eclipse, etc.).

🛠️ Clase: CalculadoraSuma

Proporciona la lógica para procesar la adición de dos sumandos.

Entrada: double a, double b 📥

Salida: double (resultado de la suma a+b) 📤
